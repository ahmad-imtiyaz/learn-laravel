| Method     | Contoh Route                                                         | Fungsi / Kegunaan                                                       |
| :--------- | :------------------------------------------------------------------- | :---------------------------------------------------------------------- |
| **GET**    | `Route::get('/hewan', [HewanController::class, 'index']);`           | Untuk **menampilkan data / halaman**                                    |
| **POST**   | `Route::post('/hewan', [HewanController::class, 'store']);`          | Untuk **mengirim data** dari form ke server (misalnya tambah data baru) |
| **PUT**    | `Route::put('/hewan/{id}', [HewanController::class, 'update']);`     | Untuk **update data** (biasanya dipakai di form edit)                   |
| **PATCH**  | `Route::patch('/hewan/{id}', [HewanController::class, 'update']);`   | Sama seperti `PUT`, tapi bisa untuk update sebagian data                |
| **DELETE** | `Route::delete('/hewan/{id}', [HewanController::class, 'destroy']);` | Untuk **menghapus data**                                                |
| **ANY**    | `Route::any('/hewan', function(){ ... });`                           | Menerima **semua jenis request** (GET, POST, dll)                       |
| **MATCH**  | `Route::match(['get', 'post'], '/hewan', ...);`                      | Hanya menerima jenis request tertentu yang kita tentukan                |
